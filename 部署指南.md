# 明月工作台 — 部署指南

## 一句话说明
你需要做的就是：注册 Vercel → 上传文件 → 得到链接。全程约 5 分钟。

---

## 步骤

### 第1步：注册 Vercel（只需一次）
1. 手机浏览器打开 https://vercel.com
2. 点 "Sign Up"
3. 选 "Continue with Email"（最简单），输入你的邮箱和密码
4. 去邮箱点验证链接
5. 注册完成

### 第2步：创建项目
1. 登录后，你会看到 Dashboard
2. 点击 "Add New..." → "Project"
3. 页面会问你导入 Git 仓库，**不用管**
4. 往下滑，找到 "Import Third-Party Git Repository" 或者直接点旁边的 "Upload" / "Deploy Template"

**最简单的方式：用拖拽上传**
1. 在 Vercel Dashboard 右上角点 "Add New..." → "Project"  
2. 跳过 Git 导入，直接往下找到上传入口
3. 把我给你的两个文件（index.html 和 vercel.json）拖进去
4. 点 "Deploy"

### 第3步：等待部署
- 通常 10-30 秒就完成了
- 完成后 Vercel 会给你一个链接，类似：
  `https://mingyue-pm.vercel.app`
- 这个链接就是你的工作台！

### 第4步：添加到手机主屏幕
1. 用 Safari 打开你的链接
2. 点底部分享按钮（方框+箭头图标）
3. 选 "添加到主屏幕"
4. 这样它看起来就像一个 App

---

## 以后如何更新
1. 我帮你改好代码后，你下载新的 index.html
2. 去 Vercel Dashboard → 你的项目 → Settings → 重新上传
3. 自动更新，链接不变

---

## 文件说明
- `index.html` — 工作台主文件（所有功能都在里面）
- `vercel.json` — 告诉 Vercel 怎么部署（不需要改动）

## 配置云同步（跨设备数据同步）

工作台支持通过 **GitHub 私有 Gist** 同步数据，数据只有你能看到。

### 第1步：创建 GitHub Token
1. 打开 https://github.com/settings/tokens/new?scopes=gist&description=mingyue-pm
2. 登录 GitHub 后直接点页面底部 "Generate token"
3. 复制生成的 Token（`ghp_` 开头，只显示一次，请保存好）

### 第2步：在工作台连接
1. 打开工作台，点击右上角 **☁** 按钮
2. 粘贴 Token → 点击"连接"
3. 首次连接会自动创建私有 Gist 并推送数据

### 第3步：在其他设备同步
1. 在新设备打开工作台链接
2. 点击右上角 **☁** 按钮
3. 粘贴**同一个 Token** → 点击"连接"
4. 自动找到已有 Gist，拉取最新数据

配置完成后，每次修改数据会**自动推送**，每次打开页面会**自动拉取**最新数据。

---

## 注意事项
- 数据同时保存在本地 localStorage 和云端 Gist（已配置云同步时）
- 清除浏览器数据后，重新配置 Token 即可从云端恢复
- Token 只存在你自己的设备本地，不会上传到任何第三方
